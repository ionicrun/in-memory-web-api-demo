## Install & Run

```bash
# download or clone
$ git clone https://github.com/ionicrun/in-memory-web-api-demo.git
$ cd in-memory-web-api-demo

# install the packages
$ npm i

# serve or run
$ ionic cordova run ios
# or
$ ionic cordova run android

```
